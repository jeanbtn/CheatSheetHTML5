Les Basiques

Post by Jean SNOW, July 07, 2016.
HTML5 BASIC

  • <!DOCTYPE> : défini le type de document
  • <html> : défini un document HTML
  • <title> : défini un titre pour le document
  • <body> : délimite le body du document
  • <h1> à <h6> : défini des titres dans le html
  • <p> : défini un paragraphe
  • <!--...--> : insérer un commentaire

Les Tableaux

Post by Jean Snow, July 07, 2016.
HTML5 TABLE

  • <table> : défini un tableau
  • <th> : défini un header dans un tableau
  • <tr> : défini une rangée dans un tableau
  • <td> : défini une cellule dans un tableau
  • <thead> : groupe un contenu dans le header du tableau
  • <tbody> : groupe un contenu dans le body du tableau
  • <tfoot> : groupe un contenu dans le footer du tableau

Styles/Sections

Post by Jean Snow, July 07, 2016.
HTML5 STYLE SECTION

  • <style> : défini un style pour un document
  • <div> : défini une section dans un document
  • <span> : défini une section dans un document
  • <header> : défini un haut de page pour un document
  • <footer> : défini un bas de page pour un document
  • <hgroup> : groupe des titres "h1" à "h6"
  • <section> : défini une section dans un document

Liens

Post by Jean Snow, July 07, 2016.
HTML5 LINK

  • <a> : défini un lien
  • <link> : défini une relation entre un document et une source extérieure
  • <nav> : défini des liens de navigations

Formes

Post by Jean Snow, July 07, 2016.
HTML5 FORM

  • <form> : défini une forme HTML pour un utilisateur
  • <input> : défini un contrôle de contribution
  • <textarea> : défini une multiligne
  • <button> : défini un bouton clickable
  • <select> : défini une liste déroulante
  • <optgroup> : défini une groupe de listes déroulantes
  • <option> : défini une option dans une liste déroulante
  • <label> : défini un label à une contribution
  • <output> : défini le résultat d'un calcul
